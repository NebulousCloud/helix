ix.faction

Helper library for loading/getting faction information.

Functions

ix.faction.Get(identifier)

 View source »

Retrieves a faction table.

Parameters

  • identifier

    Index or name of the faction

Returns

  • table

    Faction table

Example Usage

print(ix.faction.Get(Entity(1):Team()).name)
 > "Citizen"
